    Java 的构造函数的特点

    • 构造器用于初始化对象的状态(数据)
    • 构造函数不能有返回类型
    • 构造函数隐式调用
    • 如果没有指定任何构造函数，java编译器提供一个默认构造函数
    • 构造函数的名称必须和类名称相同
